# 🚨 AERG — AI Emergency Response Grid

> Reducing emergency response time using AI-powered smart dispatch and real-time coordination.

---

## 🧠 Problem Statement

Emergency response delays cost lives.  
Traditional ambulance dispatch systems lack intelligent routing, real-time coordination, and predictive insights.

In critical situations, even a few minutes can determine survival.

---

## 💡 Our Solution

AERG (AI Emergency Response Grid) is an AI-powered smart ambulance dispatch system that:

• Detects emergency incidents  
• Automatically assigns the nearest available ambulance  
• Optimizes routes in real-time  
• Provides live tracking dashboard  
• Predicts ETA dynamically  

Our goal is to reduce response time by up to 40% using intelligent automation.

---

## ⚙️ Key Features

✔ Real-time ambulance tracking  
✔ Smart AI-based dispatch logic  
✔ Dynamic ETA prediction  
✔ Live monitoring dashboard  
✔ Accident detection simulation (AI module)  
✔ WebSocket-based instant updates  

---

## 🏗️ System Architecture

**Frontend:** React 18 + Vite + Leaflet  
**Backend:** Node.js + Express + Socket.io  
**AI Module:** Python (YOLOv8 Simulation)  
**Communication:** REST APIs + WebSockets  
**Deployment:** Vercel (Frontend) + Render (Backend)

---

## 🔄 How It Works

1. Emergency incident is detected or reported  
2. AI identifies nearest available ambulance  
3. Smart routing calculates fastest path  
4. Live tracking updates in real time  
5. Dashboard monitors response status  

---

## 📊 Expected Impact

🚑 Faster emergency response  
📉 Reduced dispatch delays

🧠 Smarter decision-making through AI
